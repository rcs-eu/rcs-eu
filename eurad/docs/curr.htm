Curr in package EuradSystem

Class representing a currency.

Features

Code: String
Code of currency, e.g. PLN, USD etc.
Sign: String
Sign of currency, e.g. zł, $ etc.
GetCurr(serviceProvider:IEcoServiceProvider; currName:String; out curr:Curr):Boolean
Method. Returns true if a currency with currName code exists in the system and sets this currency to curr parameter. Otherwise returns false and sets null to curr parameter.
GetCurr(serviceProvider:IEcoServiceProvider; currName:String):Curr
Method. Gets a currency with code as currName parameter.
