CustomerOffer in package EuradSystem

Class representing an offer for a customer.

Abound()
Method. Changes an offer's state from InWork or Prepared to Abounded.
Accept()
Method. Changes an offer's state from Sent to Accepted.
AcceptIfInternal()
Method. Is fired on entry to state Sent and only for internal offers. When this offer is connected to deliverer contract (internal price list) then contract is registered upon data from this offer. When this offer is created upon customer request which is connected to a price request then automatically prices are set to a price request upon data from this this offer.

Method. Calculates and returns offer similarity rate.
CanPrepare():Boolean
Method. Checks if an offer's state can be changed from InWork to Prepared - if all data are filled and proper. Returns true if everything is O.K. and state can be changed to Prepared.
CleanExchangeRates()
Not in use.

Method. Returns offer's number. Used for stating a number of a last prepared document for a customer.
DoPrepare(ensureDelivererContractCostGroups:TriggerParamWrapper)
Method. Is executed when an offer's state is changed from InWork to Prepared and:
- checks if an internal offer does not contain internal costs,
- checks if this offer has at least 1 suboffer
- some other data for internal documents' flow.
DoSend()
Method. Is fired when an offer's state is changed from Prepared to Sent.
