AbstractContractor in package EuradSystem

Abstract class representing a contractor.

Features

FCooperationState: CooperationState
Cooperation state of a contractor. It is enum property. It can have the following values:
- Ok = 0,
- Warning = 1,
- Bad = 2.
FCooperationStateDesc: String = <NULL>
Description for cooperation state with a contractor. Connected with FCooperationState property.
FFullName: String
First line for a contractor's full name.
FFullNameSec: String = <NULL>
Second line for a contractor's full name.
FFullNameThr: String = <NULL>
Third line for a contractor's full name.
FIdentifier: String
Contractor's short identifier.
IsCustomer: Boolean
Derived. It is a flag informing if a contractor can have a customer role.
IsDeliverer: Boolean
Derived. It is a flag informing if a contractor can have a deliverer role.
IsRecipient: Boolean
Derived. It is a flag informing if a contractor can have a recipient of a cargo role.
IsSender: Boolean
Derived. It is a flag informing if a contractor can have a sender of a cargo role.
AddRole(roleType:Type):ContractorRole
Method. Adds a roleType role for a contractor and returns an added ContractorRole object.
GetHaveCorrespondAddress(ref res:Boolean)
GetRole(roleType:Type; out role:ContractorRole):Boolean
Method. Gets a contractor's role. If a contractor has a roleType role it returns true and role objects is assigned to role parameter. If a contractor has not a roleType role it returns false and null is assigned to role parameter.
GetRole(roleType:Type):ContractorRole
Method. Gets a contractor's role of a roleType role. If a contractor has a roleType role it returns a ContractorRole object for a roleType role. If a contractor has not a roleType role it throws EuradException with a message "Role roleType not found".
HasRole(roleType:Type):Boolean
Method. Returns true if a contractor can have a roleType role, otherwise it returns false.
RemoveRole(roleType:Type)
Method. Removes a roleType role from a contractor. If a contractor has not a roleType role it throws EuradException with a message "Role roleType not found".
SetHaveCorrespondAddress(value:Boolean)
