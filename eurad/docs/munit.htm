MUnit in package EuradSystem

Represents a measure unit.

Features

MLName: String = <NULL>
Stores all language-specific versions of the Name.
PredefinedUnit: PredefinedUnit = None
Determines whether this is one of the predefined measure units.
Possible values are:
0 - None,
1 - Tone,
2 - Jednostki,
3 - Waggon,
4 - Truck,
5 - Ship
EnsureUnit(ecoServiceProvider:IEcoServiceProvider; language:Language; name:String):MUnit
Method. Searches for a measure unit matching the specified name. If multiple measure units were found only the first is returned. If no measure unit was found, a new measure unit is created and returned.
GetUnit(ecoServiceProvider:IEcoServiceProvider; predefinedUnit:PredefinedUnit; out unit:MUnit):Boolean
Method. Searches for a measure unit matching the specified predefinedUnit. If multiple measure units were found only the first is returned in the unit out parameter. The return value determines whether a measure unit was found.
GetUnit(ecoServiceProvider:IEcoServiceProvider; language:Language; name:String; out unit:MUnit):Boolean
Method. Searches for a measure unit matching the specified name. If multiple measure units were found only the first is returned in the unit out parameter. The return value determines whether a measure unit was found.
