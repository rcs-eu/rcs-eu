CostGroup in package EuradSystem

Class representing a group of costs.

Features

CanEdit: Boolean
Derived. True if this group can be edited i.e. no executed items were connected to this group.
GroupType: CostGroupType
Derived. Type of group depending on kind of CostGroupDetail. Enum:
- None,
- Waggon,
- Truck,
- Ship,
- Customs,
- Gos,
- Terminal,
- Other,
- Unknown.
IsRailEmpty: Boolean
Derived. True if on a list of costs there is a cost with railway condition with empty limit. Otherwise false.
Note: String = <NULL>
Internal note.
Nr: Integer
Derived. Reference number of a group in a list of groups for a calculation.

Associations

AddCostList : costlist 1
Poins to a list of additional costs.
Calculation : offeredservicecalc 1
Points to a calculation to which this group of costs belongs.
CostGroupDetail : costgroupdetail 1
Points to an object which is a detail for this group and is specific for different type of services.
CostList : costlist 1
Points to a list of costs.
DelivererContractService : deliverercontractservice 0..1
Points to a service from a deliverer contract which is adequate to this group. It is used in contracts for internal company units.
ExecutedItems : executeditem 0..*
A list of executed items to which this group was connected.
Prices : offeredserviceprice 0..*
A list of prices which were created upon costs from this group.
