NavCloseOrder in package EuradSystem

Integration action which places information about customer order's state change to the Navision buffer. This information reflects customer order state change to 'Closed'.

Features

Execute(taskExecutionInfo:TaskExecutionInfo)
Method. Executes this integration action.

Associations

InInvoiceZero : ininvoicezero 0..1
The zero invoice which closes the customer order.
SingleCustomerOrder : singlecustomerorder 0..1
The customer order that has been closed. Will be not used.
