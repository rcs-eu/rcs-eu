Configuration in package EuradSystem

Abstract class representing a set of configuration properties.

Features

RootState: String State diagram
A configuration's state.
BeginEdit()
Method. Is triggered when a configuration enters the Edit state. Fills ChangedProperties association with AllProperties.
Edit()
Method. Changes a configuration's state from Ready to Editing.
EndEdit()
Method. Is triggered when a configuration exits the Edit state. Removes unchanged configuration properties from the ChangedProperties list.
GetPropertyValue(property:ConfigProperty):AnyValue
Method. Gets a value for a property. Returns AnyValue representing the value of the specified property.
GetPropertyValue(id:String):AnyValue
Ready()
Method. Changes a configuration's state from Editing to Ready.

Associations

AllProperties : configproperty 0..*
Derived. The configuration properties of this configuration.
ChangedProperties : configproperty 0..*
A list of configuration properties with changed values (values different from the configuration property's DefaultValue).
