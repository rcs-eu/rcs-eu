ReloadUnit in package EuradSystem

Class representing a reload unit.

Features

Note: String = <NULL>
Internal note.
PackagesAmount: Decimal
Amount packages that was reloaded.
StringRepresentation: String
Depending on what kind of cargo holder this unit has it returns an identifier of a unit. For:
- waggon it is a waggon's number,
- storage unit it is a storage unit's name,
- ship it is a ship's name,
- truck it is a registration's number.
Weight: Decimal
Weight of cargo that was reloaded.
