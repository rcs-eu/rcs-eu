12.02 ExecutedItemSearching

Placed classes

ExecutedItem
Abstract class representing an executed item which corresponds to services accomplished by deliverers for our customers. It can be registered in a customer order.
ExecutedItemRailBillFinder
Helper class for searching waggons registered in customer orders and used for automatic waggons' costs controlling.
ExecutedItemsFinder
Helper class for searching executed items registered in customer orders and used for automatic items' costs controlling.
ExecutedItemShipFinder
Helper class for searching ships registered in customer orders and used for automatic ships' costs controlling.
ExecutedItemTruckFinder
Helper class for searching trucks registered in customer orders and used for automatic trucks' costs controlling.
InvoiceSpecItem
Represents invoiced service. For InInvoices this should provide all kinds of single ExecutedItem details. For OutInvoice this should be linked with single ExecutedService.
