ExchangeRateSingleList in package EuradSystem

Class representing a list of exchange rates in which from and to currencies can be different.

Features

Representation: String = <NULL>
A whole list as one string expressed in a user-friendly way.
AddRateIfNotExists(fromCurr:Curr; toCurr:Curr; addDefaultValues:Boolean):ExchangeRateSingle
Method. If rate from fromCurr to toCurr not exists in the list then it adds such a rate. If addDefaultValues is true then it gets counter and value from NBM exchange rate table.
CopyFrom(exchangeList:ExchangeRateSingleList)
Method. Copies all exchange rates from exchangeList.
GetRate(fromCurr:Curr; toCurr:Curr):ExchangeRateSingle
Method. Gets ExchangeRateSingle for rate from fromCurr to toCurr.
Rate(fromCurr:Curr; toCurr:Curr):Decimal
Method. Returns elementary amount for rate from fromCurr to toCurr.
Recalculate(fromCurr:Curr; toCurr:Curr; value:Decimal):Decimal
Method. fromCurr value = toCurr x . Returns x
