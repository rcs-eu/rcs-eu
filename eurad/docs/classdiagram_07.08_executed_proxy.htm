Abstract class representing a holder for cargo, i.e. waggon holder, truck holder, ship holder etc.

Abstract class representing an executed item which corresponds to services accomplished by deliverers for our customers. It can be registered in a customer order.

Class representing an item which is a part of MasterItem and was created by splitting MasterItem registered in general customer order to many single customer orders.
