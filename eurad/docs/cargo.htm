KindOfService: KindOfService
Enum. Sets and gets a kind of service. It can have following values:
- None = 0: for no services,
- Rail = 1: for railway transport services,
- Truck = 2: for truck transport services,
- Ship = 4: for ship transport services,
- GOS = 8: for border services,
- TP = 16: for reloading and storage services,
- AC = 32: for customs services,
- Other = 64: for other services,
- NeedsNhm = Rail | Ship | AC: for services that require NHM,
- OptionalNhm = TP | GOS: for services that NHM is optional,
- NeedsCargoType = Truck: for services that require a cargo type,
- AllowsNhm = NeedsNhm | OptionalNhm: for services that allow setting NHM,
- AllowsCargoType = NeedsCargoType: for services that allow setting a cargo type.
