PayTermDetail in package EuradSystem

Determines the due date of an invoice or another document.

Features

Days: Integer
Number of days added to the date specified by FromDate and altered according to FromType.
FromDate: PayTermFromDate
Determines which date should be used for calculations. Possible values are:
0 - invoice date,
1 - service date,
2 - receive date.
FromType: PayTermFromType
Determines whether the due date calculation should use the exact day or the end of the month. Possible values:
0 - use exact date,
1 - use the last day of the month specified.
CopyFrom(termin:PayTermDetail)
Copies the information from the specified termin into this PayTermDetail.
GetAsText(language:String; docType:PeyTermDocType):String
Retrieves the string representation for this object.
GetPaymentDate(serviceDate:DateTime?; invoiceDate:DateTime?; resultDateMustBeWorkDay:Boolean):DateTime?
Calculates invoice due date for the specified service date and invoice date. If the result is not a work day and resultMustBeWorkDay was specified, then the next work day is returned.
