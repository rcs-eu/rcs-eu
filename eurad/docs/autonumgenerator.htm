AutoNumGenerator in package EuradSystem

Class representing an object which is responsible for generating next number for autonumeration.

Features

Enabled: Boolean = true
If true generator is enabled. Otherwise it is disabled.
EntityName: String
User-friendly name.
EntityTypeNames: String
Containes names of classes for which this AutoNumGenerator generates numbers.
StrategyData: String
Containes data of StrategyId strategy, i.e.:
- mask, i.e. {2}/{1:yyMM}/{0:00000},
- last generated number for a strategy.
StrategyId: String
Specifies how auto numbers are generated. It can have following values:
- KeyMaskedStrategy: sequence is never restarted,
- MonthMaskedStrategy: each month has a seperate sequence,
- YearMaskedStrategy: each year has a seperate sequence.
UseParentGenerator: Boolean = false
If true generator from company unit's parent is taken to generate next number. Otherwise this generator is taken to generate next number.
