03.01 System Messages

Placed classes

AbstractRoot
The root class for most of the model classes.
AttachDocument
Class representing an attachment document which is attached to a message in a message system.
Attachment
Abstract class representing attachment attached to a message in a message system.
Company
Class representing an outer contractor.
CustomerOffer
Class representing an offer for a customer.
CustomerRequest
Class representing a request for an offer from a customer.
GeneralDelivererOrder
Class representing a general order to a deliverer.
GosCargoHolder
Class representing cargo holder in GOS registry.
InInvoice
Represents invoice received from a deliverer.
Message
Abstract class representing base of message in message system.
MessageBySystem
Predefined Message in message system. Mostly used when system needs to report some object action or state change.
MessageCrmComercialLimitChanged
Message sent when commercial limit for Company is changed.
MessageCrmCooperationStateChanged
Message sent when cooperation state for Company is changed.
MessageCustOfferNoAnswer
Message sent when there is no answer from a customer on CustomerOffer.
MessageCustReqNew
Message sent to a user when there is a new internal CustomerRequest which involves a user.
MessageCustReqRespUserChanged
Message sent to a user who was set as a responsible user for a CustomerRequest.
MessageDelOrderGenAccepted
Message sent when internal deliverer accepts Order.
MessageDelOrderGenInternalNew
Message sent to internal deliverer when a new general deliverer Order was sent to them.
MessageDelOrderGenRejected
Message sent when an internal general deliverer Order is rejected.
MessageDelOrderSinAccepted
Message sent when an internal single deliverer Order is accepted.
MessageDelOrderSinExecuted
Message sent when an internal single deliverer Order was executed.
MessageDelOrderSinInternalNew
Message sent to internal deliverer when a new single deliverer Order was sent to them.
MessageDelOrderSinRejected
Message sent when an internal single deliverer Order was rejected.
MessageGosCargoHolderConnectedToOrd
Message sent when a GosCargoHolder from a GOS registry was connected to a single customer order meaning that a service is realized.
MessageGosCargoHolderNewAviso
Message sent when a GosCargoHolder from a GOS registry was connected to a general customer order which is treated as a arrival aviso.
MessageInInvInternalNew
Message sent when a new internal incoming Invoice was registered.
MessageInInvNewCbrImport
Message sent when a new import of CBR incoming Invoice were done.
MessageInInvUnbooked
Message sent when an icoming Invoice's state was changed from "Booked" to "Closed".
MessageOutInvUnbooked
Message sent when an outgoing Invoice's state was changed from "Booked" to "Sent".
MessagePriceReqAnswered
Message sent when an anwswer for internal PriceRequest was registered.
MessagePriceReqNoAnswerFromCustomer
Message sent when there is no answer from a deliverer for a PriceRequest within a specified days.
MessagePriceReqRejected
Message sent when an internal PriceRequest was rejected.
OutInvoice
Abstract class representing an outgoing invoice for a customer.
PriceRequest
Price request to deliverer.
SingleDelivererOrder
Class representing single deliverer order
