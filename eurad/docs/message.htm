Message in package EuradSystem

Abstract class representing base of message in message system.

Features

LinksStr: String
Derived. String describing how to display all of the attachments as link (eg. document number, file name ect.)
MessageDateTime: DateTime
Specifies date when message is intended to be sent by system.
MessageDescription: String
Derived. Needs to be handled in child class.
MessageText: String
Derived. Needs to be handled in child class.

Associations

Attachments : attachment 0..*
Message attachments as list of Attachment objects.
Deliveries : messagedelivery 1..*
As Message can be sent to more than one recipient - this lists contains all required MessageDelivery log.
Documents : abstractroot 0..*
List of objects (eg. CustomerOrder, OutInvoice, etc.) message is bound to.
