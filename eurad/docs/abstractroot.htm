AbstractRoot in package EuradSystem

The root class for most of the model classes.

Features

CreatedOn: DateTime
Records the moment of creation of this object.

Method. Adds observer to the list of objects receiving change notifications.
DeriveMLString(fieldName:String):MLString
DetachObserver(observer:AbstractRoot)
Method. Removes observer from the list of objects receiving change notifications.
HasCurrUserPriv(priv:String; inUnit:SysCompanyUnit):Boolean
NotifyObservers(value:AbstractRoot)
Method. Notifies all observers about a change of this object. Calls each attached observer's ObserverUpdate() method passing value as a parameter.
ObserverUpdate(value:AbstractRoot)
Method. Notifies about a change in the observed object (there can be only one observed object at a time).
