IntegrationTask in package EuradSystem

Represents named integration task which can be executed.

Features

Executed: Boolean
Indicates whether this task has been successfully executed.
Name: String
The name of the task.
QueuedAt: DateTime
The date and time when this task has been queued for execution.
Execute(taskExecutionInfo:TaskExecutionInfo)
Method. Executes the task. A task can only be executed if Executed is false.

Associations

Actions : integrationaction 0..*
Ordered list of actions for this task. All actions must be successfully executed for the task to be successfully executed.
Job : integrationjob 0..*
A list of integration jobs.
Queue : integrationqueue 0..1
The queue of this task.
QueuedByUser : sysuser 1
Points to the user who queued this task.
