CostValue in package EuradSystem

Class representing a cost's value.

Features

Discount: Decimal
Derived. EfectiveNet minus Net.
DiscountStr: String
Derived. Discount expressed in a user-friendly way.
EfectiveNet: Decimal
Derived. Net value of a cost after all increases and discounts.
Net: Decimal
Net value of a cost without any increases and discounts.
AddIncrease(name:String; isPercent:Boolean; value:Decimal; generatedByCostSource:Boolean):Increase
Method. Adds a discount/increase to Increases list of this cost's value.
Clear(keepUserIncreases:Boolean)
Method. Sets Net to zero. If keepUserIncreases is false then it deletes all objects in Increases list.
