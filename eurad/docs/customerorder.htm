DoAccept(genDelOrdersSender:TriggerParamWrapper)
 Method. Is executed when an order's state is changed from InWork to Accepted. It does a couple of things:
- checks for user's privileges for accepting orders from a temporary customer,
- checks for user's privileges for accepting order where a payer is temporary,
- checks if internal order does not have internal costs,
- creats required deliverer orders.
