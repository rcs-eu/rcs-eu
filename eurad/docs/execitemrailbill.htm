ExecItemRailBill in package EuradSystem

Class representing an executed item of railway type which can be registered in a customer order.

Features

CalculatedWeight: Decimal
Derived. Total real weight from a raillway bill taking into account weight roundings set in a customer offer which is connected to order in which this executed item is registered.
ClearKnownQuantities()
CreateInvoiceSpecItem():SpecItemInInv
Method. Returns an invoice's specification item of RailwayBill type created upon this executed item.
GetLine1():String
Method. Returns waggon count or waggon's number as a string.
GetLine2():String
Method. Returns dispatch date as a string.
GetLine3():String
Method. Returns NHM numbers of transported cargo as a string.
GetLine4():String
Method. Returns total real weight as a string.
GetLine5():String
Method. Returns a dispatch station's name.
GetLine6():String
Method. Returns a destination station's name.
GetOffersToConnect():IObjectList
Method. Returns a list of offers suitable for this executed item.
GetOrderedServicesToConnect():IDictionary<OrderedService, CostGroup>
Method. Returns a list of services from an order suitable for this executed item.
MeetsPriceCondition(priceCondition:PriceCodition):Boolean
Method. Checks if data from RailwayBill meets all conditions from priceCondition. It is used to find a proper single price or cost for this executed item.
SetKnownQuantities()
Method. Sets quantity in a QList:
- for PredefinedUnit.Waggon it sets waggon count,
- for PredefinedUnit.Jednostki it sets waggon count or 1 if waggon count equals 0,
- for PredefinedUnit.Tone it sets CalculatedWeight.
TryConnectToGosCargoHolder()
Method. Tries to connect this executed item to a corresponding waggon from GOS registry.
