AnyValue in package EuradSystem

Abstract class representing user configuration's property value.

Features

Clone():AnyValue
Method. Clones this object and returns the clone.
IsEqual(value:AnyValue):Boolean
Method. Checks if this object is equal to value object. Returns true if this object is equal to value. Otherwise returns false.

Associations

DefaultForProperty : configproperty 0..1
PropertyValue : propertyvalue 0..1
Points to the optional PropertyValue this AnyValue is value of. Either PropertyValue or DefaultForProperty should have value.
