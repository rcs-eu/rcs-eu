CustomerRequest in package EuradSystem

Class representing a request for an offer from a customer.

Features

IncomeDate: DateTime
Date of incoming of this request.
IsInternal: Boolean
Derived. True if this request is from internal customer. Otherwise false.
Note: String = <NULL>
Internal note.
RegisterDate: DateTime
Date of registration of this request.
RejectNote: String = <NULL>
Note which is filled when request's state is changed to Rejected. This note has a reason of rejection.
RootState: String State diagram
Request's state.
CalculateSimilarityRate(similarOffer:ISimilarOffer):Int64
Method. Calculates and returns request similarity rate.
Cancel()
Method. Changes a request's state from InWork to Canceled.
ConnectToOffer()
Method. Chagnes a request's state ffom Prepared to Offered.
DisconnectFromOffers()
Method. Changes a request's state from Offered to Prepared.
DocNr():String
Method. Returns request's number. Used for stating a number of a last prepared document for a customer.
DoReject()
Method. Is executed on entry to state Rejected. If this request is internal this method changes a state of connected price request from Sent to Unanswered.
GetDocDate():DateTime
Method. Returns request's registration date. Used for stating a date of a last prepared document for a customer.
GetListOfSimilarOffers():ISimilarOffer[]
Method. Returns a list of offers and customer requests which are similar to this request.
GetOwningCompany(out company:Company):Boolean
Method. Returns true if this request's customer is an outer company and sets this company to company parameter. Otherwise returns false and sets null to company parameter.
GetParts():SimilarOfferPart[]
Method. Gets groups of costs from this customer request as parts which are used in searching for similar offers and requests.
MarkAsInWork()
Method. Changes a request's state from Prepared to InWork.
MarkAsPrepared()
Method. Changes a request's state from InWork to Prepared.
MarkAsPreparedEffect()
Method. Is executed when a request's state is changed from InWork to Prepared. It checks for similar requests and offers.
Reject()
Method. Changes a request's state from InWork to Rejected.
