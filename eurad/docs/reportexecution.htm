ReportExecution in package EuradSystem

Keeps information of single report generation.

Features

GenerationStart: DateTime
The moment when the generation of the report started.
GenerationStop: DateTime = <NULL>
The moment when the generation of the report stopped.
GenerationTime: TimeSpan = <NULL>
Derived. Report generation time. This is the time span between GenerationStart and GenerationStop.

Associations

LastUserReportExecution : lastuserreportexecution 0..1
Report : report 1
The report that was executed.
UsedOptions : usedoption 0..*
A list of parameter values used during report generation.
User : sysuser 1
The user of the report.
