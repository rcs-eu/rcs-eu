ChangeLog in package EuradSystem

Features

ChangeAt: DateTime
Date and time of a change.
ChangedFrom: String = <NULL>
An old value of changed property.
ChangedProp: String = <NULL>
Changed property.
ChangedTo: String = <NULL>

ChangeText: String
Derived. User-friendly text stating what kind of a change were made.
DescOfChanges: String = <NULL>
Description for a change.
ObjId: String
ID of an object in which a property was changed.
ObjNr: String = <NULL>
Number of a document in which property was changed.
ObjType: String
A class of object in which property was changed.
