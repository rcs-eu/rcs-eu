AdvancedOfferPriceCalculator in package EuradSystem

Class representing an object which is used for automatic calculation of offered service's prices. It calculates prices for railway services having costs and conditions for price.

Features

RecalculatePrice()
Method. Recalculates offered service's prices. It's executed every time costs change.

Associations

ExchangeRateSingleList : offerexchagelist 1
Points to a table of exchange rates which is used for automatic calculation of prices.
OfferedService : offeredservice 1
Points to an offered service for which prices have to be automatically calculated.
