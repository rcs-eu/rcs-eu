DelivererContractService in package EuradSystem

Specifies a contracted service.

Features

IsDiscount: Boolean
Determines whether this service prices' are expressed as a discount from a rail tariff.
Clone():DelivererContractService
Method.

Associations

The contract item for this contract service.
Detail : costgroupdetail 1
Holds the details of this service.
DiscountList : discountlist 1
Points to the price list for this service expressed as a list of discounts from a rail tariff. Is null if IsDiscount is false.
OfferCostGroup : costgroup 0..1
If this contract service's contract is internal, then OfferCostGroup points to the customer offer's service on wich this contract service is based. Otherwise it is null.
OfferedService : offeredservice 0..1
If this contract service's contract is internal, then OfferedService points to the customer offer's service on wich this contract service is based. Otherwise it is null.
PriceList : pricelist 1
Points to the price list for this service expressed as fixed price values. Is null if IsDiscount is true.
