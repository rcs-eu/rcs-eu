CbrImportConfig in package EuradSystem

Class representing an object which holds configuration of CBR invoices' import.

Features

GetConfig(serviceProvider:IEcoServiceProvider):CbrImportConfig
Static method. Gets an object of this class.
GetSysCompanyFor(sysUnitCode:String):SysCompanyUnit
Method. Gets internal company unit from SysUnits list with sysUnitCode. If company unit was not found then it throws ApplicationException with a message "Company unit for this code not found!".
GetSysCompanyFor(sysUnitCode:String; out unit:SysCompanyUnit):Boolean
Method. Gets internal company unit from SysUnits list with sysUnitCode. Returns true if company unit was found and sets this found company unit to unit parameter. Returns false if company unit was not found and sets null to unit parameter.
GetVatFor(vatCode:String; out vat:Vat):Boolean
Method. Gets VAT from VatCodes list with vatCode. Returns true if VAT was found and sets this found VAT to vat parameter. Returns false if VAT was not found and sets null to vat parameter.
GetVatFor(vatCode:String):Vat
Method. Gets an object of Vat class from VatCodes list with vatCode. If VAT was not found then it throws ApplicationException with a message "VAT seting for this code not found!".
