03.00 Messaging

Placed classes

AbstractRoot
The root class for most of the model classes.
AttachDelOrderGeneralInternal
Class representing an object that is created as an attachment document in a message system when message regarding general order for internal deliverer is created.
AttachDelOrderSingleInternal
Class representing an object that is created as an attachment document in a message system when message regarding single order for internal deliverer is created.
AttachDocument
Class representing an attachment document which is attached to a message in a message system.
AttachFile
Class representing an attached file.
AttachGosCargoHolder
Class representing an object that is created as an attachment document in a message system when a message regarding registering a new waggon in GOS is created.
Attachment
Abstract class representing attachment attached to a message in a message system.
Message
Abstract class representing base of message in message system.
MessageBySystem
Predefined Message in message system. Mostly used when system needs to report some object action or state change.
MessageByUser
Class representing a message sent by a user to another user.
MessageDelivery
Class representing a delivery of a Message.
MessageType
Not in use.
SysUser
Represents a user of the system.
