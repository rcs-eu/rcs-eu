RailwayBill in package EuradSystem

Class representing a railway bill.

Features

BillDocCIM: Byte[] = <NULL>
A file of CIM document prepared in Jerid system.
BillDocSMGS: Byte[] = <NULL>
A file of SMGS document prepared in Jerid system.
ContractNr: String
Contract't number according to which this transport is executed.
DispatchDate: DateTime
Dispatch date.
FIsEmpty: Boolean
True for empty transport if a registered bill reflects a group of waggons without a waggon list.
FOwner: WaggonOwner
An owner of all waggons if a registered bill reflects a group of waggons without a waggon list. Enum:
- All = 0 (both private and railway),
- Railway = 1,
- Privat = 2.
FTotalAxles: Integer
Total number of axles. Used when there is a rail transport without a waggon list.
FTotalRealWeight: Decimal
Total real weight if a registered bill reflects a group of waggons without a waggon list.
Gauge: LoadingGauge
Derived. Loading gauge. If this bill contains waggons and all waggons have the same Gauge then gauge of first waggon is returned. If this bill reflects a group of waggons without a waggon list then it is LoadingGauge.Any.
Enum:
- Any = 0,
- Normal = 1,
- WM0 = 2,
- WM1 = 3.
HasRealWeight: Boolean
HasUti: Boolean
Derived. True if waggons from this bill has containers.
HasWaggons: Boolean
Derived. True if this bill contains waggons.
IsEmpty: Boolean
Derived. If this bill contains waggons or containers then it is true if all waggons or containers are empty. It this bill reflects a group of waggons without a waggon list then it is FIsEmpty.
IsSingleWaggon: Boolean
True if bill contains only one waggon.
IsWaggonGroup: Boolean
Derived. True if a registered bill has a list of waggons or it reflects a group of waggons without waggon list.
LoadLength: Decimal
Derived. Loading length. If this bill contains waggons and all waggons have the same LoadLength then loading length of first waggon is returned. If this bill reflects a group of waggons without a waggon list then it is 0.
Owner: WaggonOwner
Derived. If this bill contains waggons then it is an owner of all waggons. If all waggons are private then it is private. If all waggons are railway then it is railway. If there are both private and railway then it is all. If this bill reflects a group of waggons without a waggon list then it is FOwner. Enum:
- All = 0 (both private and railway),
- Railway = 1,
- Privat = 2.
RailwayBillType: RailwayBillType
Bill's type. Enum:
- Unspecified,
- CIM,
- SMGS,
- CETAR.
TotalAxles: Integer
Derived. If there are waggons in this bill then it is a sum of their axles. If there are no waggons in this bill then it is equal to FTotalAxles.
TotalRealWeight: Decimal
Derived. If this bill contains waggons or containers it is a total real weight of waggons or containers. Otherwise it is FTotalRealWeight.
WaybillNr: String = <NULL>
Bill's number.
