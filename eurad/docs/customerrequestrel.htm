CustomerRequestRel in package EuradSystem

Class representing a service in a customer request.

Features

Nr: Integer
Derived. Reference number of a service in a list of services for a request's item.
RelationFrom: String
Derived. States what is a starting station for transport in this service.
RelationTo: String
Derived. States what is a ending station for transport in this service.
ServiceName: String = <NULL>
Service's name.

Associations

CustomerRequestItem : customerrequestitem 1
Points to a request's item.
OfferedService : offeredservice 0..1
Points to a service from an offer which was created upon this request's service.
PriceRequestSeg : pricerequestseg 0..1
If this request was created from an internal price request it points to a corresponding service from a price request.
Segments : custreqcostgroup 0..*
A list of cost groups.
