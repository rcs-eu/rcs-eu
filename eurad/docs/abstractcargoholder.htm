AbstractCargoHolder in package EuradSystem

Abstract class representing a holder for cargo, i.e. waggon holder, truck holder, ship holder etc.

Features

CargoHolderId: String
Derived. Id of a cargo holder:
- waggon's number for CargoHolderWaggon class,
- registration plate's number for CargoHolderTruck class,
- ship's name for CargoHolderShip class,
- container's number for CargoHolderUti class,
- storage unit's name for CargoHolderStorageUnit.
CargoHolderType: String
Derived. Type of a cargo holder:
- "Wagon" for CargoHolderWaggon class,
- "Samochód" for CargoHolderTruck class,
- "Statek" for CargoHolderShip class,
- "UTI" for CargoHolderUti class,
- "Magazyn" for CargoHolderStorageUnit class.
RealWeight: Decimal
