RailTariff in package EuradSystem

A separate functional unit which represents a rail tariff. It can be executed by the main part of the application.

Features

ValidFrom: DateTime
Specifies the first day of validity of the tariff.
ValidTo: DateTime
Specifies the last day of validity of the tariff.
FindTariff(serviceProvider:IEcoServiceProvider; name:String; validOn:DateTime; out tariff:RailTariff):Boolean
Static method. Returns the tariff with the specified name in tariff. The tariff must be valid at the specified validOn date. The result determines whether the requested tariff was found.
GetInstance(user:SysUser):RailTariffInstance
Method. Instantiates a new tariff instance. RailTariffInstance provides specific report generation functionality.
