ExecItemShip in package EuradSystem

Class representing an executed item of ship type which can be registered in a customer order.

Method. Checks if data from Ship meets all conditions from priceCondition. It is used to find a proper single price or cost for this executed item.

Method. Sets quantity in a QList:
- for PredefinedUnit.Ship it sets 1,
- for PredefinedUnit.Jednostki it 1,
